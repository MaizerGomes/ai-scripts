# AI Usage Conductor

A bash application to check installed AI CLI tools, fetch their usage/quota information, and display it in a unified dashboard.

## Features
- Detect installed AI CLI tools (mods, aichat, sgpt, ollama, etc.).
- Fetch token/cost usage from provider APIs (OpenAI, Anthropic, Gemini).
- Parse local usage history from tool databases (mods, aichat).
- Unified terminal dashboard UI.
