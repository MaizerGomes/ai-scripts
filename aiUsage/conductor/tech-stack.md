# Tech Stack

- **Language:** Bash (Zsh compatible)
- **Utilities:**
  - `curl`: API requests.
  - `jq`: JSON processing.
  - `sqlite3`: Parsing local databases (mods, aichat).
  - `tput`: UI formatting.
  - `bc`: Calculations (for cost estimation).
- **Optional Tools:**
  - `gum`: For interactive UI elements (if available).
